# 🌍 Proyecto destacado

## ⛽ GasoApp  
🔗 https://gasoapp.es

**GasoApp** es una aplicación web que permite encontrar **las gasolineras más baratas de España en tiempo real** utilizando datos oficiales del gobierno.

### Funcionalidades

- Consulta precios de **más de 12.000 gasolineras en España**
- Datos oficiales del **Ministerio de Industria (MITECO)**
- Búsqueda por **ubicación, provincia o municipio**
- Visualización en **lista o mapa interactivo**
- Interfaz optimizada para **móvil**

### Para usuarios registrados

- Guardar **gasolineras favoritas**
- Registrar **repostajes y gastos**
- Crear **alertas de precio**
- **Calculadora de combustible**

### Características técnicas

- Actualización automática de datos cada **24 horas**
- **Notificaciones push** para alertas de precio
- **GitHub Actions** ejecuta un cron cada 6 horas para comprobar alertas
- Aplicación desplegada en producción
- Desarrollo asistido con herramientas de IA para acelerar el aprendizaje y la implementación

### Stack utilizado

- SvelteKit  
- Supabase  
- Vercel  
- TypeScript  
- TailwindCSS  
- Web Push (VAPID)
